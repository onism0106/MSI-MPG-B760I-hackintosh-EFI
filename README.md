# MSI-MPG-B760I-hackintosh-EFI

OC：0.9.5
CPU:13600KF
显卡RX6600
内存:DDR5 32GB *2
硬盘:西数SN850X


支持Ventura/Sonoma

BIOS设置：
1.关闭安全启动
2.关闭CFG Lock
3.开启D.T.M.（可选）

wifi蓝牙正常
睡眠正常，不会自动唤醒

存在的问题：唤醒后蓝牙无法使用,但是有解决方案
具体现象：
唤醒后蓝牙自动关闭，手动开启后搜索不到设备，且无法再开启或关闭

解决方案:睡眠前关闭蓝牙,唤醒后打开蓝牙,可以使用自动化工具来完成:
教程:http://www.manongjc.com/detail/42-zzaflsxoovdzvts.html
